# 飞机边界线图绘制软件 (Python版本)

这是一个用于绘制飞机边界线图的软件，使用Python和PyQt5重新开发，功能与原Qt C++版本完全相同。

## 功能特点

- 支持不同类型飞机参数选择：通航飞机（单发/双发）、农用飞机、双发涡桨飞机
- 支持不同发动机类型选择：活塞、涡桨
- 根据输入参数计算并绘制飞机边界线图
- 用户友好的图形界面

## 安装说明

1. 确保您的系统已安装Python 3.6或更高版本

2. 安装依赖库：

```bash
pip install -r requirements.txt
```

## 运行方法

直接运行main.py文件启动应用程序：

```bash
python main.py
```

## 使用说明

1. 在主界面输入各项参数：有效载荷、巡航速度、巡航高度等
2. 点击"选择飞机类型"按钮，在弹出的对话框中选择飞机类型
3. 点击"选择发动机类型"按钮，在弹出的对话框中选择发动机类型
4. 点击"计算"按钮，软件将计算并在下方图形区域绘制飞机边界线图

## 主要参数说明

- 有效载荷（kg）：飞机能够携带的有效载荷重量
- 巡航速度（m/h）：飞机在巡航阶段的速度
- 巡航高度（m）：飞机在巡航阶段的飞行高度
- 飞行待机用油（s）：飞行待机时需要的燃油量
- 航程（km）：飞机的最大航程
- 起飞场长（m）：起飞所需的场地长度
- 着陆场长（m）：着陆所需的场地长度
- 最大升力系数：飞机的最大升力系数
- 设计爬升率（m/s）：飞机的设计爬升率
- 失速速度（m/s）：飞机的失速速度
- 初始估算重量（kg）：飞机的初始估算重量
- 升阻比：飞机的升力与阻力之比
- 展弦比：飞机的翼展与翼弦之比
